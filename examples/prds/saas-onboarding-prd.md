# PRD: SaaS Onboarding Revamp

## Context
Our onboarding completion rate dropped from 62% to 48% over two quarters.

## Problem
New users struggle to understand setup requirements in the first session.

## Goals
- Increase onboarding completion to 65% in 8 weeks.
- Reduce first-session drop-off by 20%.
- Improve setup task clarity.

## Requirements
- Add checklist-based onboarding flow.
- Add progress indicator and completion nudges.
- Include in-app setup guidance for integrations.

## Success Metrics
- Completion rate
- Time to first key action
- Week-1 retention
